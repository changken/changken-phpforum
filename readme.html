如何安裝？

1.解壓縮檔案。
2.新增MySQL資料庫。
3.將upload內的檔案上傳。
4.在瀏覽器網址欄打http://你的網站網址/install。
5.按照安裝導引進行安裝。
6.刪除install檔案夾。
7.登入控制後台，選網站參數設定，將參數填妥。
8.即可使用。
註1:控制後台管理員帳號可在安裝導引中新增。

如何安裝外掛？

請注意！
安裝非官方發佈之外掛會有一定之風險！
請斟酌安裝使用！
 1.請詳閱外掛說明書。
2.安裝外掛完畢後，請定時注意外掛作者是否有釋出更新！

更新changken-phpforum或外掛

更新changken-phpforum或外掛請務必備份網站資料和MySQL資料庫！
 如何更新主程式？
1.請將新版本資料上傳覆蓋至網站根目錄(請注意！一定要先備份！)
 2.選擇更新檔(例:upgrade-35to40.php)，上傳至網站根目錄。
3.執行更新檔。
4.顯示更新完成後，就更新完成了！

如何更新外掛？
1.請參閱各外掛說明書。
